puppet-module-rpcbind

Puppet module to manage rpcbind

To use this module, simply include ::rpcbind

Compatibility

This module is built for use with the latest versions of Puppet v4 and v5 with the ruby version packaged in the all-in-one installer. See .travis.yml for the supported matrix of versions.

Supports the following platforms.

  • Debian 8
  • Debian 9
  • EL 6
  • EL 7
  • Suse 11
  • Suse 12
  • Ubuntu 14.04 LTS
  • Ubuntu 16.04 LTS
  • Ubuntu 18.04 LTS
